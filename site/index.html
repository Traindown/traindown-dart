Traindown

A language to help athletes express their training inspired by Markdown.

Traindown was created to help you easily capture all the things you do in a training session. It aims to provide you with a flexible, easy to remember language for expressing your training for record keeping purposes.

The easiest way to explain what that all means is by showing you. If you've never written a line of code in your life, don't worry! Writing out your training in Traindown is as easy as using a hashtag.

The Basics

The minimum viable Traindown file could be expressed as 

Squat: 500

This is saying "I did one rep of 500 in the Squat." 500 of what, you may ask? Well, that brings us to the next section!

Metadata

Traindown provides tools for recording units (and any other data you can imagine) with metadata. Metadata are just key value pairs that begin with a #. You can have as much or as little metadata as you prefer, but the more the better in my opinion!

Let's add some metadata to the session from above to help us understand more about what happened that day.

# Unit: lbs
Squat: 500

Ah, that's better. So this session used pounds for the unit of weight. That squat was 500 pounds.

Metadata allows you to express facts about the context of your training. Some common keys are date, time, duration, day/week of training cycle, and if you have any injuries you are working through. Some folks capture things like their weight that day or even what they eat leading into it. Let's go ahead and some more data to our example session.

# Date: Dec 25 2019
# Time: 8am
# Burritos: 3
# Unit: lbs

Squat: 500

By adding more details in the metadata, you can make more sense of the why in your training. Metadata is also very searchable, so if you choose to use a Traindown enabled analysis tool, you can really start to make sense of the big picture.

Sets and Reps

Now, our example is pretty unrealistic. Cold squatting 500 pounds, three burritos or none, is not common. Let's assume this is a more mortal trainee and fill in some of the details of the day.

# Date: Dec 25 2019
# Time: 8am
# Burritos: 3
# Unit: lbs

Squat: 135 10r 2s 225 5r 315 5r 405 2r 455 500 3s

That's more realistic. So what was added here? It should read intuitively but to be sure, let's dive into the details.

Looking at

135 10r 2s

we can read this as "I did two sets of ten reps at 135 pounds on the Squat." As you can see, we denote sets with an s and reps with an r.

Further on down the day, we can see

225 5r

which can be read as "I did one set of five reps at 225 pounds on the Squat." If we omit either an r or and s, the default is 1. So circling back to the original 500, it could be expressed as 

500 1r 1s
Note: Don't do this. Save yourself the 1r / 1s shenanigans.

but the whole point of Traindown is to make recording your training easy, not annoying.

Formatting for Reading

Back to the example...I personally like how succinct it looks but it is difficult to read, especially after three burritos. The same training could be expressed in a more reader-friendly format like

# Date: Dec 25 2019
# Time: 8am
# Burritos: 3
# Unit: lbs

Squat:
  135 10r 2s
  225 5r
  315 5r
  405 2r
  455
  500 3s

I dunnno about you, but that is much easier on my eyes.

Notes

So now we have a good idea of what happened on that day, but I'd imagine after eating three burritos, they may have been some...issues along the way. Lucky for us, Traindown provides tools for capturing notes. Let's add some!

# Date: Dec 25 2019
# Time: 8am
# Burritos: 3
# Unit: lbs

* Definitely should have stopped at two burritos.

Squat:
  135 10r 2s
  225 5r
  315 5r
  405 2r
  455
  500 3s

What we just added was a session note. This is a note that applies to the entire training session. These notes are helpful for painting a landscape of your training. Let's add some fine details! 

# Date: Dec 25 2019
# Time: 8am
# Burritos: 3
# Unit: lbs

* Definitely should have stopped at two burritos.

Squat:
  135 10r 2s
  225 5r
  315 5r
  405 2r
    * Had to stop at two reps to run to the bathroom.
  455
  500 3s
    * Surprisingly easy singles given that episode at 405.
    * Consider going up in weight next week.

Each combination of a load for sets and reps is considered a performance of a given movement. Notes that appear after a performance are called...you guessed it, performance notes. You can apply to whatever granularity of work you like to capture. For example, the three singles at 500 could look something like 

  500
    * Surprisingly easy single given that episode at 405.
  500
    * Whoops. Spoke too soon. Clenching!
  500
    * Another easy rep.

It's entirely up to you how you want to record your training and I recommend trying out different strategies to see what feels best for you.

FAQs

What if part of my training is done in a different unit?

Ah, a true connoisseur of units, I see. This is well supported using metadata and the "Unit for" key. For example, lets say we needed to add some kettlebell swings to our session from above.

# Date: Dec 25 2019
# Time: 8am
# Burritos: 3
# Unit: lbs

* Definitely should have stopped at two burritos.

Squat:
  135 10r 2s
  225 5r
  315 5r
  405 2r
    * Had to stop at two reps to run to the bathroom.
  455
  500 3s
    * Surprisingly easy singles given that episode at 405.
    * Consider going up in weight next week.

# Unit for Kettlebell swings: kg
Kettlebell swings: 24 10r 4s

Metadata can be injected into any spot in a Traindown file. Here, we added it right above the movement. We could have just as easily added it to the metadata at the top of the file. I find adding it right above the exercise helpful, but feel free to do it wherever.

What if I happen to miss a lift?

Bummer. But it happens to everyone. If you are training and need to capture failed attempts, Traindown allows you to do that with the "zero rep" 

# Unit for Snatch: kg
Snatch:
  60 3s
  80 2s
  100
  110
  120 0r
    * Shoe came off.
  120

Here we can see that the lifter was successful up until the 120 at which point they happened to miss the first attempt there due to a footwear malfunction. They appear to have been able to locate the errant shoe and rally to successfully hoist 120.

How do I capture the concept of a superset?

Blasting your bis and tris, I see? Well, suns outs, guns out! Traindown supports chaining movements together using the + modifier prepended to the names of the subsequent movments. Let's try it out 

Bi Blaster 9000: 100 20r 3s
  * This blasted my arms.

+ Tri Destroyer Supreme:
  90 10r
  100 10r
  110 10r
    * I finally ascended to 110. BRB posting to Instagram.

You can chain as many movements together as you feel like doing. The number of sets should match but if you happen to want another set on the Tri Destroyer Supreme, go for it.

Does Traindown support non-weight based training?

Sadly, we can't lift weights all the time. Sometimes we wind up doing things like cardio or maybe even compete in sports that don't involve weights. Traindown is flexible enough to help you record all kinds of training. Let's see an example—this one is for a sprinter. 

# Date: 12/25/19

Warmup drills: 3s

# Unit for 60m sprints: seconds
60m sprints: 7.12 7.03 7.01

# Unit for Jog: meters
Jog: 1600

Units in Traindown can really be anything, so whatever it is you are doing, you likely can record it with Traindown.
